◆ MIKMIDIPacketCreateFromCommands()

MIDIPacket* MIKMIDIPacketCreateFromCommands ( MIDITimeStamp  timeStamp,
MIKArrayOf(MIKMIDICommand *) *  commands 
)

Creates and returns a pointer to an MIDIPacket struct containing the given commands. This function dynamically allocates its result, and can create packets containing more than 256 bytes.

Note
: You must free the resulting MIDIPacket pointer when you are done with it by calling MIKMIDIPacketFree().
Parameters
timeStampThe timestamp for the commands in the resulting packet. Individual timestamps in the command objects themselves are ignored.
commandsAn array of MIKMIDICommand objects to include in the result.
Returns
A pointer to MIDIPacket struct.
