15 /**
16  * MIKMIDIDestinationEndpoint represents a source (input) MIDI endpoint.
17  * It is essentially an Objective-C wrapper for instances of CoreMIDI's MIDIEndpoint class
18  * which are kMIDIObjectType_Destination type endpoints.
19  *
20  * MIDI destination endpoints are contained by MIDI entities, which are in turn contained by MIDI devices.
21  * MIDI messages can be outputed through a destination endpoint using MIKMIDIDeviceManager's
22  * -sendCommands:toEndpoint:error: method.
23  *
24  * Note that MIKMIDIDestinationEndpoint does not declare any methods of its own. All its methods can be
25  * found on its superclasses: MIKMIDIEndpoint and MIKMIDIObject. Also, MIKMIDIDestinationEndpoint itself
26  * is only used to represent MIDI endpoints owned by external applications/devices. To create virtual
27  * destination endpoints to be owned by your application and offered to others, use its subclass,
28  * MIKMIDIClientDestinationEndpoint instead.
29  *
30  * @see -[MIKMIDIDeviceManager sendCommands:toEndpoint:error:]
31  * @see MIKMIDIClientDestinationEndpoint
32  */
33 @interface MIKMIDIDestinationEndpoint : MIKMIDIEndpoint <MIKMIDICommandScheduler>
