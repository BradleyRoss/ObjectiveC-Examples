48 /**
49 * Convenience method to create a note off command from a note on command with zero velocity,
50 * or another note off command.
51 *
52 * Some MIDI devices send a note on message with zero velocity when a key or button is released,
53 * instead of sending a note off command. Writing code to deal with this possibility can be
54 * somewhat ugly, especially in Swift. Using this method, a note on command can be "transformed"
55 * into a note off command if its velocity is zero with a single call, then a single execution path
56 * to handle note off commands can be written.
57 *
58 * Note that this method returns nil if the passed in command has a velocity greater than zero.
59 * @param note An instance of MIKNoteCommand.
60 * @return An instance of MIKMIDINoteOffCommand with the same note, channel, and timestamp as note.
61 * nil if note is a note on whose velocity is greater than zero.
62 */
63 + (instancetype _Nullable)noteOffCommandWithNoteCommand:(MIKMIDINoteCommand *)note;
