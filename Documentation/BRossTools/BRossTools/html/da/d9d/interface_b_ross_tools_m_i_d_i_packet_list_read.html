Parse a MIDIPacketList object.

When used with processPacket method, this method will parse a MIDIPacketList and the MIDIPacket objects contained within it.

This code should handle multiple messages per packet. Some SysEx commands should be processed even it they are contained in the packet with other types of commands. However, there should be warnings since that would be an illegal format.
