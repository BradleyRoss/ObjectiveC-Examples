Detailed Description

Construct a MIDIPacketList object.

The following are some of the structures used for MIDIPacketList,

  • struct MIDIPacketList, and MIDITimeStamp
    • UInt32 numPackets
    • MIDIPacket packet[1]
  • struct MIDIPacket
    • MIDITimeStamp timeStamp
    • UInt16 length
    • Byte data[256]
  • UInt64 MIDITimeStamp

The following are the constants for the command codes.

See https://www.midi.org/specifications-old/item/table-1-summary-of-midi-message

kMIDICVStatusNoteOffvalue = 0x8, 2 data bytes
kMIDICVStatusNote On value = 0x9, 2 data bytes
kMIDICVStatusPolyPressurevalue = 0xA, 2 data bytes
kMIDICVStatusControlChangevalue = 0xB, 2 data bytes
kMIDICVStatusProgramChange value = 0xC, 1 data byte
kMIDICVStatusChannelPressurevalue =0xD, 1 data byte
kMIDICVStatusPitchBend value = 0xE, 2 data bytes holding a 14 bit integer
A value of 15 indicates non musical commands value = 0xF, variable number of data bytes
