15 /**
16  * MIKMIDIObject is the base class for all of MIKMIDI's Objective-C wrapper classes for CoreMIDI classes.
17  * It corresponds to MIDIObject in CoreMIDI.

20  * to all MIDI objects.

83 /**
84  * Indicates whether the object is "virtual". This has slightly different meanings
85  * depending on the type of MIDI object.
86  *
87  * For MIKMIDIDevices, virtual means that the device does not represent a MIDIDeviceRef.
88  * Virtual devices can be used to wrap virtual, deviceless endpoints created
89  * e.g. by other software, some Native Instruments controllers, etc.
90  *
91  * For MIKMIDIEntitys, virtual means that the entity is part of a virtual device
92  * and its endpoints are virtual endpoints.
93  *
94  * For MIKMIDIEndpoints, virtual means that the endpoint is a virtual endpoint,
95  * rather than an endpoint of a (non-virtual) MIDI device.
96  *
97  * @seealso -[MIKMIDIDeviceManager virtualSources]
98  * @seealso -[MIKMIDIDeviceManager virtualDestinations]
99  */
