CoreMIDI

I ran the C header file for the CoreMIDI framework through Doxygen and posted the results here. I have been comparing the Doxygen information with the Apple Developer Documentation for Core MIDI. It appears that understnding the CoreMIDI framework requires looking at multiple reference sources.

Some items are used in the examples I found on the internet, but are currently deprecated. The articles on the internet are old, but I can't find newer examples.

In the header file defining structs for MIDIEventList and MIDIPacketList, it states

  • typedef struct MIDIEventList MIDIEventList; // Universal MIDI Packet
  • typedef struct MIDIPacketList MIDIPacketList; // Legacy MIDI 1.0 Packets (deprecated)

However, MIDIPacketList is not listed as deprecated in the developer documentation. (search for "API_DEPRECATED" in the header files for other items to be reviewed.

CoreMIDI Examples from Internet

I have been searching the internet for examples of Objective-C code using the CoreMIDI library. The biggest problem is that I have had a problem finding recent examples, with most of them being over ten years old. It appears that many of these examples use API's that are deprecated and will require updating.
