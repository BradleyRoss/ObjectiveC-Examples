MIDI Software Libraries

This is the start of a document discussing a number of software libraries that can be used to create MIDI software. It will be enlarged as I get more information. Except for my own software, I can't really comment on the software quality or current state of maintenance. However, I will try to flesh these out as I get more information.

Apple MacOS

  • Apple CoreMIDI Documentation The amount of documentation available for MIDI 2.0 modules is extremely limited, but I am hopeful that the information in https://imitone.com/discover-midi/ is proving very useful in understanding the Apple documentation.
  • Working with MIDI on Mac (OS X)
  • mixedinkey-opensource/MIKMIDI, GitHub project It's not clear to what extent this represents the current code base. However, it appears that it is worth looking at. More documentation can be found here. From what I can see, work is proceeding on adapting the software to ARM64 architecture, but I'm not sure of the status of this work.
  • The BRossTools Xcode project in the GitHub repository bradleyross/ObjectiveC-Examples contains a few attempt to write software using the Apple Core MIDI framework. Some of the demonstrations in this package are updated versions of some other examples that I found in the literature. This is very definitely a work in process, but some of the code may be useful to programmers.
