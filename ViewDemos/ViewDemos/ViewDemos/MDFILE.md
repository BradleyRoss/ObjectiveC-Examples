#  ViewDemos
The purpose of this project is to look at a number of types of layouts for Cocoa, such as

* [NSStackView](https://developer.apple.com/documentation/appkit/nsstackview?language=objc) 
* [NSGridView](https://developer.apple.com/documentation/appkit/nsgridview?language=objc) 
* [NSTabView](https://developer.apple.com/documentation/appkit/nstabview?language=objc)
* [NSTableView](https://developer.apple.com/documentation/appkit/nstableview?language=objc)
* [NSOutlineView](https://developer.apple.com/documentation/appkit/nsoutlineview?language=objc)
* [NSBrowser](https://developer.apple.com/documentation/appkit/nsbrowser?language=objc)
* [NSCollectionView](https://developer.apple.com/documentation/appkit/nscollectionview?language=objc)
* [NSSplitView](https://developer.apple.com/documentation/appkit/nssplitview?language=objc)
